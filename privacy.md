# 隐私政策 / Privacy Policy

最后更新日期：2026-07-08

---

## 中文（简体）

**1. 我们收集的信息**

Hope（盼头）是一款主要在您本机（Windows 桌面）运行的轻量级时间管理工具。除下方第 2 条说明的必要网络请求外，我们不会收集您的姓名、邮箱、手机号、账号或精确地理位置等个人身份信息，也不会出售或共享您的数据。您创建的任务、截止时间、颜色、循环规则等数据仅保存在本机，不会上传至任何服务器。

**2. 本应用发起的网络请求**

本应用仅在以下情况下连接网络：

- **检查更新**：向官方更新服务器发送应用版本号与操作系统信息，用于判断是否有新版本。不含任何可识别个人的内容。
- **匿名使用统计**：通过第三方分析服务 Aptabase（us.aptabase.com，美国政府区云）上报匿名使用事件，用于了解装机量、日活跃、版本与语言分布。每次请求携带技术字段：系统语言、操作系统名称与版本、应用版本、会话标识（每次启动随机生成，非稳定设备 ID）及事件时间；其中更新相关事件另含版本号、更新来源等技术属性。我们**不采集任何个人身份信息**，也不会生成持久的设备标识符。网络请求本身会包含您的设备 IP 地址，Aptabase 仅做必要处理，不用于识别个人。

**3. 数据存储与安全**

您的任务与设置保存在本地，由您自行掌控；卸载应用后相关数据随之清除。Aptabase 所收集的匿名统计数据无法关联到您个人。

**4. 您的选择**

您可在应用设置中关闭「发送匿名使用统计」。关闭后我们将不再向 Aptabase 发送任何遥测事件；检查更新功能不受影响，仍会在您主动检查时发起（仅发送版本与系统信息）。

**5. 儿童隐私**

本应用不面向儿童收集个人信息。

**6. 政策变更**

若未来功能变化导致数据收集方式改变，我们将更新本政策并告知用户。

**7. 联系我们**

如有疑问，可通过 GitHub 仓库 https://github.com/CooloiStudio/Hope 联系开发者。

## English

**1. Information We Collect**

Hope is a lightweight time-management tool that runs primarily on your local device (Windows desktop). Except for the necessary network requests described in Section 2, we do not collect your name, email, phone number, account, or precise geolocation, and we do not sell or share your data. The tasks, deadlines, colors, and recurrence rules you create are stored only on your local device and are never uploaded to any server.

**2. Network Requests Made by the App**

The app connects to the network only in the following cases:

- **Update check**: sends the app version and operating system information to our official update server to determine whether a newer version is available. Contains no personally identifiable information.
- **Anonymous usage statistics**: reports anonymous usage events to a third-party analytics service, Aptabase (us.aptabase.com, US government-region cloud), to understand install counts, daily active users, and version/language distribution. Each request carries technical fields: system locale, OS name and version, app version, and a session identifier (randomly generated on each launch, not a persistent device ID), plus the event timestamp; update-related events additionally include technical attributes such as version tag and update source. We collect no personally identifiable information and generate no persistent device identifier. The network request inherently includes your device IP address, which Aptabase processes only as necessary and does not use to identify you.

**3. Data Storage and Security**

Your tasks and settings are stored locally and are under your control; they are removed when you uninstall the app. The anonymous statistics collected by Aptabase cannot be linked to you as an individual.

**4. Your Choices**

You can disable "Send anonymous usage statistics" in the app settings. Once disabled, we will not send any telemetry events to Aptabase; the update check remains available and still sends only version and system information when you check manually.

**5. Children's Privacy**

This app does not collect personal information from children.

**6. Changes to This Policy**

If future feature changes alter how data is collected, we will update this policy and notify users.

**7. Contact Us**

For questions, contact the developer via the GitHub repository https://github.com/CooloiStudio/Hope.
